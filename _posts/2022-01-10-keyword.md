## swift 키워드

### 선언에 사용되는 키워드
- import: 모듈(프레임워크)을 로드하여 현재 파일에서 심볼을 접근할 수 있게함.
- class: 클래스 선언
- protocol: 프로토콜 선언
- extenion: 클래스 확장
- typealias: 기존타임에 별칭을 부여함
- enum: 열거형 선언
- struct: 구조체 선언
- init: 생성자
- deinit: 소멸자
- func: 함수정의
- var: 변수 정의에 사용하는 키워드
- let: 상수 정의에 사용하는 키워드
- static: 해당 타입의 모든 인스턴스가 공유하는 type 속성, 메소드 정의를 위한 키워드
- subscript: 클래스, 구조체, 열거형의 속성을 인덱스 문법으로 접근하기 위한 인터페이스
- internal
- public
- private
- inout: 함수 내부엥서 변경된 인자값이 원래 변수값에 반영되야 할 경우 사용하는 키워드
- operator

### 문장에 사용되는 키워드
- switch
- case
- break
- fallthrough: 코드 제어흐름을 다음 case 문으로 이동
- default
- contionue
- if
- else
- where
- for
- in
- do
- while
- return
- repeat: while문과 조합하여 1번 이상의 반복문을 수행
- defer: 실행 흐름이 현재 코드블럭을 떠나기 전에 특정 문장집합을 실행하고 싶을 때 사용
- guard: 에러조건이 발생하여 메서드를 조기 종료하고 에러를 보고 하기 위해 사용함

### 표현식과 타입
- is
- as
- dynamicType: 인스턴스 타입, 런타임 시점에 타입을 반환함
- super
- self
- _COLUMN_: 시작열 번호를 나타내는 리터럴
- _FILE_: 파일의 이름을 나타내는 리터럴
- _FUNCTION_: 선언부의 이름을 나타내는 리터럴
- _LINE_: 시작라인 번호를 나타내는 리터럴
- try
- throws
- rethrows
- catch
